基于 LangChain4j AiServices 构建 Agent 智能客服

背景

在本地生活闪购平台 LocalMind 中，用户经常咨询重复性问题——“我的订单状态是什么””这个优惠券还有库存吗””附近有没有火锅店”。每次都需要人工客服介入，效率低下。

技术选型

  • LLM: 阿里云百炼 Qwen-Max
  • Agent 框架: LangChain4j 0.31.0
  • 工具定义: @Tool 注解
