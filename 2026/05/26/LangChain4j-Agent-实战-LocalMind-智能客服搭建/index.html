手写 LangChain4j Agent：为本地生活平台搭建智能客服

发表于 2026-05-26 分类于 项目实战

项目背景

在 LocalMind —— 一个本地生活优惠抢购平台中，用户每天会产生大量重复性咨询：

  • “我的订单什么时候到？”
  • “XX 店的优惠券还能抢吗？”
  • “附近有没有评分高的火锅店？”

如果全靠人工客服，效率低且成本高。于是决定用 LangChain4j + 阿里云百炼 Qwen-Max 构建一个 AI Agent 来处理这些场景。

技术选型

组件 选型 理由
LLM 阿里云百炼 Qwen-Max 国内可用、中文能力好、支持 Function Calling
Agent 框架 LangChain4j 0.31.0 Java 生态原生支持、@Tool 注解简洁
对话记忆 MessageWindowChatMemory 内置窗口记忆，保留最近 20 轮上下文
会话隔离 ConcurrentHashMap 每个 sessionId 独立维护一个 Agent 实例
